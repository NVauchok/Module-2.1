Task 2

Use the scenarios created in scope of the task 2.1, 3.1, 3.2. Add data reading from a database.
Preparation:
-	Download and install SQL Management studio under the following path: \\epam.com\sam\SoftArchive\DBA\Microsoft\SQL Management Studio Express\2012
-	Download and install SQL Server from: http://www.microsoft.com/ru-ru/download/details.aspx?id=29062 
-	Download install package of Northwind database under the link: http://www.microsoft.com/en-us/download/details.aspx?id=23654 
-	Install the package (Northwind database will be available under the folder c:\SQL Server 2000 Sample Databases\ after installation)
-	Log in to the SQL management studio express with your domain credentials and attach Northwind database. To attach after log in to the sql management studio right click over ‘Databases’ item -> Attach -> Select Northwind.mdf file from the folder c:\SQL Server 2000 Sample Databases\
-	Database is ready to work. You can check database please and description here. http://northwinddatabase.codeplex.com 
Implementation:
-	Process different kinds of requests as SELECT, DELETE, UPDATE, INSERT using ADO.NET (do not use the requests described into the presentation and examples)
-	Call at least one stored procedure (except CustOrderHist which is used in presentation) using ADO.NET
